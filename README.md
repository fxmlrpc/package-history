**SAX serialization logic for FXMLRPC.**


## Install

Via Composer

``` bash
$ composer require fxmlrpc/sax-serialization
```


## Usage

This is the SAX (using XmlReader and XmlWriter) implementation of serialization logic used in FXMLRPC.


## Testing

``` bash
$ composer test
```
